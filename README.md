# Data Structures Visualizer

A Python GUI application that visualizes fundamental data structures including Stack, Queue, and Linked List. The program allows users to perform operations interactively and view how the data structures change step by step.

## Features

• Stack operations (Push and Pop)  
• Queue operations (Enqueue and Dequeue)  
• Linked List operations (Insert Node and Delete Node)  
• Operation history tracking  
• Simple GUI built using Tkinter  
• Beginner-friendly visualization of data structures  

## Technologies Used

Python  
Tkinter

## How to Run

1. Download or clone the repository

git clone https://github.com/yourusername/data-structures-visualizer.git

2. Navigate to the project folder

cd data-structures-visualizer

3. Run the program

python datastructuresvisualizer.py

## Project Objective

The objective of this project is to help users understand how common data structures work by providing an interactive graphical interface where operations can be performed and observed in real time.

## Learning Outcomes

• Understanding of Stack, Queue, and Linked List operations  
• Practice with Python GUI development using Tkinter  
• Implementation of basic data structure algorithms
